
2A - Travaux dirigés - Deep Learning
------------------------------------

Exercices pour deep learning pour le cours :ref:`l-td2a`.
Les notebooks qui vont de 100* à 300* sont des notebooks
d'introduction avec `pytorch <http://pytorch.org/>`_
qui fonctionne sous Linux.
Il est préférable d'avoir du GPU (plus de cent fois
plus rapide que du CPU).
