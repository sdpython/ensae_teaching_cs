Format du fichier à fournir

Le fichier que doit fournir le candidat est un fichier answer.txt avec une seule colonne qui contient la probabilité de défault de paiement pour chaque utilisateur de la base d'évaluation. Le fichier n'a pas d'entête. Exemple : 

0.67
0.11
0.4678
...

Le fichier est soumis à Codalab après qu'il a été zippé. Exemple : answer.zip.
