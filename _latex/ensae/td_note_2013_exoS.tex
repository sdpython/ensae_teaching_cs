Il est présent à la fin de l'énoncé de l'exercice. \textbf{Afin d'éviter son inclusion (et son impression), votre programme devra impérativement commencer par~:}

\begin{verbatimx}

Pour visualiser la matrice et obtenir la figure~\ref{double_spirale_2013}, il faut écrire~:
\begin{verbatimx}
exoS.dessin_matrice(matrice)   # cette fonction place un point bleu pour une case contenant 1,
                               # rouge pour une case contenant 2,
                               # rien si elle contient 0
\end{verbatimx}

\exequest On a tous les éléments pour écrire l'algorithme de coloriage~:
\begin{enumerate}
\item On part d'un point $(i0,j0)$ qui fait partie de la zone à colorier, qu'on insère dans une liste \codes{acolorier = [(i0,j0)]}.
\item Pour tous les points $(i,j)$ de la liste \codes{acolorier}, on change la valeur de la case $(i,j)$ de 0 à 2.
\item Pour tous les points $(i,j)$ de la liste \codes{acolorier}, on regarde les quatre voisins $(i,j+1), (i,j-1), (i+1,j), (i-1,j)$. Si la matrice contient 0 pour un de ces voisins, on l'ajoute à la liste et on retourne l'étape précédente tant que cette liste n'est pas vide.
\end{enumerate}
